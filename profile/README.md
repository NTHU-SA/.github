# Hi, we are NTHU-SA

國立清華大學學生會（National Tsinghua University Student Association），是全校性的學生自治組織，扮演學生和學校間的溝通橋樑。我們傾聽不同聲音、接納多元的看法，共創理想校園。  
在這裡，我們專注於打造能實際改善清華校園生活的系統與平台，涵蓋學生服務、校園資訊、資料工具與開發者協助。

## 開發中專案
- [**NTHU-Data-API**](https://github.com/NTHU-SA/NTHU-Data-API) — 將繁雜的校園資訊重新整合，協助校園開發者開發的清華校園資料 API。
- [**NTHU-Data-Scraper**](https://github.com/NTHU-SA/NTHU-Data-Scraper) — 校園單位資料自動化整合與擷取工具，整合校園各單位的最新公告資訊。
- [**Voting-System**](https://github.com/NTHU-SA/Voting-System) — 為學生自治投票開發的投票系統，讓學生能快速、安全、匿名的投出珍貴的一票。
- [**outdoor-stage-booking**](https://github.com/NTHU-SA/outdoor-stage-booking) — 野台借用系統，提供學生線上、便捷的野台借用管道。
- [**ccxpLite**](https://github.com/NTHU-SA/ccxpLite) – 校務資訊系統插件，提供更現代的瀏覽體驗以及驗證碼自動填寫。
